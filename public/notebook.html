ARIMA-GARCH Thesis Notebook Render

Static preview of the implementation behind the master thesis “Evaluating the ARIMA-GARCH Model’s Accuracy Across Diverse Cryptocurrencies” (BTC, ETH, DOGE, SOL).

Dataset setup

Daily Yahoo Finance data from 2020-05-11 to 2024-04-20. Pipeline: clean prices, compute log returns, then split into train/validation/test with a 70/15/15 ratio.

Model selection

Automated ARIMA + GARCH-family tuning (GARCH/GJR/FIGARCH/EGARCH) using AIC and parameter significance checks. Heavy tails handled via Student’s t errors.

Evaluation protocol

Multi-horizon forecasts (h = 1/3/7/14/30) plus adaptive rolling backtests (60-day window, robustness check with 365-day window). Benchmarks: naive return forecast and EWMA volatility. Statistical checks: Diebold-Mariano, Kupiec, Christoffersen.
